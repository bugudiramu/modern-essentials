# GEMINI.md - Shared Configuration

## Overview
Shared configuration for ESLint, Prettier, Tailwind, and TypeScript.

## Contents
- `eslint-base.js`: Base ESLint rules.
- `prettier.config.js`: Shared Prettier configuration.
- `tailwind.config.js`: Master Tailwind theme and configuration.
- `tsconfig.base.json`: Base TypeScript configuration.

## Usage
- Apps and packages extend these configurations to maintain consistency across the monorepo.
